# How To Use?

1. Clone Repositories With [GIt Bash](https://git-scm.com/downloads)
<br>
```git clone https://github.com/ca-kraa/dashboard-admin.git```

2. Open Project With Visual Studio Code

3. Open Terminal
<br>

```Composer Install```

<br>
Or

<br>

```Composer Update```

4. Enjoy

Source [AdminLTE](https://adminlte.io/)
